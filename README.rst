MD2DOCX
=======

Installation
------------
Required packages:
  - python-docx (to convert md to docx)

Note that this program uses its own naive conversion from md to docx.

Optional packages:
  - PIL (to convert png to tiff)

Required programs:
  - inkscape (to convert svg images to png)
  - node.js (to prepare the citations)
